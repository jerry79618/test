我們導入哪些工具

CORE🤖

GitHub Copilot CLI

終端機裡的 AI 代理，讀規格、生測案、跑指令、自我修復，全程自然語言驅動

CORE🥒

Cucumber + Gherkin

BDD 行為驅動開發框架，Given / When / Then 中英文皆可撰寫，測案變成活文件

CORE🧠

Multi-Agent 協作

  • 需求分析 Agent — 解析 JIRA 需求單
  • 測案生成 Agent — 產生 Cucumber 測試案例
  • 程式碼 Agent — 產生 JAVA 測試程式碼
  • Code Review Agent — 代碼審查
  • 自動化測試 — 執行測試

Cucumber-JVM + Java

HTML Report

🔄

CI/CD Pipeline

整合進部署流程,每次 commit 自動回歸,上線前就把問題攔下來
